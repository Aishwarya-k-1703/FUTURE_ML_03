# Customer Support Chatbot – Task 3 (Future Interns)

## Project Description
This project is a Customer Support Chatbot developed using Dialogflow and deployed on Telegram.  
The chatbot answers common customer queries such as order status, return policy, working hours, and contact support.

This project was completed as part of Machine Learning Task 3 under the Future Interns Internship Program.

---

## Objectives
- Build a chatbot for customer support
- Handle frequently asked questions (FAQs)
- Provide greeting messages
- Handle unknown queries using fallback responses
- Deploy the chatbot on Telegram
- Test the chatbot using Dialogflow and Telegram

---

## Tools and Technologies Used
- Dialogflow
- Telegram Bot API
- Google Account
- Web Browser

---

## Features
- Greeting message
- FAQ handling (order status, return policy, working hours, contact support)
- Smart fallback response
- Telegram integration
- Real-time responses
